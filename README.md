# HHU FSCS MensaMonitor

A monitor to show what the canteens at the Heinrich-Heine-University in 
Duesseldorf offer today (Campus Vita, The Great Mensa, Piggy Mensa).

## Dependencies
* Java (8)
* JavaFX
* Gradle

## How to compile/run
Encoding: UTF-8
```
gradle build
gradle run
```
